# File Type Icons

This directory uses **Phosphor Icons** - flexible icon family by Phosphor.

- License: MIT
- Source: [phosphoricons.com](https://phosphoricons.com)
- Weight: Regular

## Icon Mapping

| File | Usage |
|------|-------|
| `folder.svg` | Directories |
| `file.svg` | Unknown/default files |
| `image.svg` | Images (jpg, png, gif, etc.) |
| `video.svg` | Videos (mp4, mov, etc.) |
| `audio.svg` | Audio (mp3, wav, etc.) |
| `pdf.svg` | PDF documents |
| `document.svg` | Word docs (doc, docx, etc.) |
| `spreadsheet.svg` | Spreadsheets (xls, csv, etc.) |
| `presentation.svg` | Presentations (ppt, key, etc.) |
| `text.svg` | Text/Markdown files |
| `code-*.svg` | Source code by language |
| `config.svg` | Config files (json, yaml, etc.) |
| `terminal.svg` | Shell scripts |
| `database.svg` | Database files |
| `archive.svg` | Archives (zip, tar, etc.) |
| `package.svg` | Packages (dmg, pkg, etc.) |
| `font.svg` | Font files |
| `3d.svg` | 3D models |
| `design.svg` | Design files |
| `ebook.svg` | eBooks |
| `lock.svg` | Dotfiles & lock files |
